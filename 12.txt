Отлично. Тогда спроектируем **полную карту 3D-лаборатории**, чтобы это можно было реально собрать в Blender и потом превратить в WebGL-сайт через Three.js.
Идея — **пространство знаний**, где пользователь буквально ходит по курсу. 🧠🎛️

---

# План 3D-лаборатории курса

Представь комнату примерно **8×6 метров**.
В ней несколько зон.

```
┌──────────────────────────────┐
│         POSTER WALL          │
│   (roadmap + архитектура)    │
│                              │
│ SHELVES         VISUAL LAB   │
│ (modules)       (screen)     │
│                              │
│          MAIN DESK           │
│      modular + laptop        │
│                              │
│    PATCH TABLE / BLUEPRINT   │
└────────────ENTRANCE──────────┘
```

Каждая зона — **раздел курса**.

---

# 1. Entrance — стартовая точка

Когда пользователь заходит на сайт:

он появляется **у входа в лабораторию**.

Перед ним:

* логотип **Modular Genesis**
* кнопка **Start Exploration**

На полу можно сделать линию-маршрут.

```
Start → Foundations → Sequencing → Generative
```

---

# 2. Главный стол (центр)

Это **сердце лаборатории**.

На столе:

* модульный синт
* ноутбук
* MIDI-клавиатура
* чертежи

```
Desk
 ├ modular synth
 ├ laptop
 ├ midi keyboard
 ├ patch notes
```

Интерактивность:

| объект     | действие          |
| ---------- | ----------------- |
| модуль     | открывает урок    |
| ноутбук    | Ableton workflow  |
| клавиатура | MIDI lesson       |
| чертеж     | архитектура патча |

---

# 3. Стена с плакатами

Тут размещаются **твои схемы**.

Например:

* архитектура курса
* генеративная система
* modular pipeline

Каждый плакат:

```
poster_foundations
poster_generative
poster_hybrid
```

Клик → открывает страницу урока.

---

# 4. Полки с модулями

Это **раздел синтеза**.

Полки можно заполнить:

* VCO
* VCF
* VCA
* LFO

Когда пользователь нажимает на модуль:

включается **демо звука**.

Например:

```
click oscillator
↓
plays saw wave
```

Это очень крутой интерактив.

---

# 5. Patch Table

Отдельный стол с:

* патч-кабелями
* схемами
* книгами

Здесь раздел:

```
Generative patches
```

При клике:

открывается:

* VCV patch
* explanation

---

# 6. Visual Lab

В углу комнаты — **экран визуализации**.

На экране:

* shader
* geometry nodes
* particles

Он реагирует на звук.

```
audio
↓
visual system
```

Это раздел курса:

```
Audiovisual systems
```

---

# 7. Blueprint wall

Стена с архитектурой системы.

На ней можно разместить схемы:

```
Clock
↓
Random
↓
Sequencer
↓
Synth
```

Это **раздел generative theory**.

---

# 8. Скрытые элементы (пасхалки)

Можно добавить:

* кабель → открывает patch
* книгу → открывает документацию
* лампу → включает ambient

Это делает сайт **живым**.
