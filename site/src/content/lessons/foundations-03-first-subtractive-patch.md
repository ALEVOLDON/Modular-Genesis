## Theory

Subtractive synthesis starts with a harmonically rich source and removes or shapes parts of it.

```text
Oscillator -> Filter -> VCA -> Output
```

This patch teaches the most durable modular ideas:

- source
- shaping
- loudness control
- output staging

## Why This Patch Matters

Almost every future system in the repository still contains this logic somewhere inside it, even when the patch becomes more complex.

## Patch Steps

1. Add an oscillator.
2. Send it into a filter.
3. Send the filter into a VCA.
4. Send the VCA into audio output.
5. Add an envelope to control the VCA.
