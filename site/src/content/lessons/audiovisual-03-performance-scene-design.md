## Theory

Performance design is the layer where all previous tracks meet:

- modular patching
- DAW routing
- controller input
- visual response

## Scene Structure

A usable scene often includes:

- one stable timing source
- one or more evolving voices
- one visual system with defined mappings
- one small set of performance controls

## Why It Matters

Without scene design, a patch may be interesting but not performable.

## Practice

Sketch a live scene with:

1. sound engine
2. control layer
3. visual layer
4. output path
