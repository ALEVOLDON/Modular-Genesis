## Theory

A patch becomes musical when parameters change over time.

Two of the main time-based control sources are:

- envelopes
- LFOs

### Envelope

An envelope reacts to a trigger or gate and shapes an event over time.

### LFO

An LFO repeats cyclic movement over time.

It is useful for:

- filter sweeps
- vibrato
- tremolo
- slow texture motion

## Patch Idea

```text
Envelope -> VCA
LFO -> Filter cutoff
```

## Practice

Make one short percussive sound and one long evolving sound using both control sources.
