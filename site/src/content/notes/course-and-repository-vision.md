## Project Intent

The repository is not only meant to hold patches. It is meant to grow into:

- an open course
- a public reference library
- a structured patch archive
- a site for modular and audiovisual education

## Educational Direction

The main levels are:

1. Foundations
2. Sequencing
3. Generative Systems
4. Hybrid Setup
5. Audiovisual Performance

This progression maps cleanly to both documentation folders and site sections.

## Lesson Format

A stable lesson format appears throughout the project:

- theory
- patch diagram
- real patch file
- short demo
- task or exercise
