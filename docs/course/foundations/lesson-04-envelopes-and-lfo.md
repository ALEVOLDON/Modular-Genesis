# Lesson 04: Envelopes And LFO

## Theory

A patch becomes musical when parameters change over time.

Two of the main time-based control sources are:

- envelopes
- LFOs

### Envelope

An envelope usually reacts to a trigger or gate and shapes an event:

- attack
- decay
- sustain
- release

### LFO

An LFO repeats cyclic movement over time.

It is useful for:

- filter sweeps
- vibrato
- tremolo
- slow motion in textures

## Patch Idea

```text
Envelope -> VCA
LFO -> Filter cutoff
```

This is the beginning of animation inside the patch.

## Practice

Make one short percussive sound and one long evolving sound:

1. use the envelope to define the event shape
2. use the LFO to create repeated motion
3. compare fast vs slow modulation
